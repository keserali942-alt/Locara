# Locora Roadmap

## Phase 0 - Infrastructure and Standards

Completion criteria:
- App launches and shows an empty home screen
- Supabase client is initialized using env variables
- Feature-first + Clean Architecture folder structure exists
- Riverpod providers are wired
- Dark mode can be toggled
- No secret is hardcoded in source files

## Phase 1+

Future phases should start only after Phase 0 completion criteria are validated.

## Phase 1 - Identity and Discovery Profile

Completion criteria:
- User can register/sign in with email and OAuth providers
- User completes onboarding survey
- Data is persisted into `users` and `discovery_profile`

Status:
- Implemented in code, waiting runtime verification in environment with Flutter SDK.
